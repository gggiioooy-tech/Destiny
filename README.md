# 15월 길드전 사이트 실행 방법

이 압축파일은 Vite + React + Tailwind 프로젝트 기본 폴더입니다.

## 1. App.jsx 붙여넣기
`src/App.jsx` 파일을 열고 기존 내용을 전부 지운 뒤,
ChatGPT 캔버스에 있는 최신 코드 전체를 복사해서 붙여넣으세요.

## 2. 설치
터미널에서 이 폴더로 이동한 뒤:

```bash
npm install
```

## 3. 실행
```bash
npm run dev
```

브라우저에서:
```text
http://localhost:5173
```

## 4. 폰 테스트
같은 와이파이에서 터미널에 뜨는 Network 주소로 접속하세요.

## 5. 배포
GitHub에 이 폴더를 올리고 Vercel에서 Import Project 하면 됩니다.
